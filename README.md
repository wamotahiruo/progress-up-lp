# Progress Up LP

「進捗アップ会」のランディングページプロジェクトです。

## 開発の始め方

```bash
# 依存関係のインストール
npm install

# ローカルサーバーの起動
npm run dev
```

## ビルドとデプロイ

本番環境向けにビルドするには以下のコマンドを実行します。

```bash
npm run build
```

`dist` ディレクトリに静的ファイルが生成されます。

### デプロイ方法

このプロジェクトは静的サイトとして、以下のサービス等に簡単にデプロイできます。

- **Vercel / Netlify**:
  - GitHubリポジトリと連携するか、`dist` フォルダをドラッグ＆ドロップしてデプロイします。
  - ビルドコマンド: `npm run build`
  - 出力ディレクトリ: `dist`

- **GitHub Pages**:
  - リポジトリの設定から GitHub Pages を有効にします。
  - または `gh-pages` パッケージを使用してデプロイします。
